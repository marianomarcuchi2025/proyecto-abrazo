# Código de Conducta

Este proyecto adopta el [Contributor Covenant v2.1](https://www.contributor-covenant.org/es/version/2/1/code_of_conduct/).

## Resumen

- Tratá a todas las personas con respeto y empatía.
- No se tolera acoso, lenguaje despectivo ni ataques personales.
- Este proyecto trabaja con y para niños vulnerables: las discusiones sobre
  usuarios finales deben ser especialmente cuidadosas y nunca incluir datos
  reales de menores (nombres, fotos, casos identificables).
- Las críticas técnicas son bienvenidas; deben dirigirse al código, no a
  las personas.

## Reportes

Comportamientos inaceptables pueden reportarse abriendo un issue o
escribiendo al mantenedor: marianomarcuchi@gmail.com. Los reportes se
tratan con confidencialidad.

## Aplicación

El mantenedor puede editar o rechazar comentarios, commits, issues y PRs
que violen este código, y bloquear temporal o permanentemente a quien
reincida.
